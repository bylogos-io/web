# Planes y Precios

Ofrecemos soluciones flexibles que se adaptan a la escala y necesidades de tu operación industrial.

## Nuestros Planes

### EDGE

**Desde 15 UF**
Solución local para control crítico donde la latencia y la autonomía son primordiales.

- Protocolos Industriales nativos.
- Hardware Optimizado Seeed Studio.
- Baja Latencia para control en tiempo real.

### CLOUD

**Desde 8 UF/mes**
Escalabilidad e inteligencia avanzada en la nube para supervisión global.

- Respaldo Híbrido de datos.
- IA Copilot para asistencia técnica e insights.
- Dashboards Web accesibles desde cualquier lugar.

### SOLUCIONES A MEDIDA

¿Necesitas algo específico? Contáctanos para una solución Enterprise.

- Soporte 24/7 dedicado.
- Integraciones personalizadas con sistemas ERP/MES.
- Infraestructura de grado empresarial.
