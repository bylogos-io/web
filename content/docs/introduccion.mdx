# ¿Qué es LogOS?

**LogOS** es la nueva manera de eficiencia industrial. Es una plataforma de integración **AI + IIoT** (Artificial Intelligence + Industrial IoT) diseñada para la optimización y modernización de infraestructuras críticas.

Nuestra misión es ser el puente entre los mundos de IT (Tecnología de la Información) y OT (Tecnología de la Operación), modernizando sistemas legados y unificando el control bajo una única plataforma mediante Inteligencia Artificial y computación Edge.

## Características Principales

- **Procesos Inteligentes**: Optimización continua mediante algoritmos avanzados.
- **Probado en Industrias Críticas**: Estabilidad garantizada en entornos exigentes.
- **Cumplimiento Normativo**: Acoplado a normativas RIC para SEC.

## Compatibilidad Técnica

LogOS está diseñado para ser agnóstico del hardware pero altamente optimizado para plataformas robustas.

### Requisitos Mínimos

- **Arquitectura**: ARM64 (Soporte x86_64 próximamente)
- **RAM**: 2GB mínimo
- **Sistema Operativo**: Linux Kernel 5.4+

### Protocolos Industriales Soportados

LogOS habla tu idioma industrial nativo:

- **IEC 61850**: Estándar global para automatización de subestaciones.
- **Modbus**: Comunicación serial robusta.
- **OPC-UA**: Interoperabilidad y modelado de datos.
- **MQTT**: Integración ligera para IoT.

## Hardware Verificado

Aunque LogOS puede correr en diversos entornos Linux, está oficialmente optimizado y probado en:

### Raspberry Pi

- Soporte completo para **Raspberry Pi 4 y 5**.
- Estado: **Optimizado**.

### Seeed Studio (Industrial)

Asociación estratégica para hardware de grado industrial.

1.  **ReTerminalDM**
    - _Uso_: Panel de Control Interactivo.
    - _Features_: Pantalla táctil 10.1", IP65, RS485.
    - _Ideal para_: Salas de control y HMI local.

2.  **ReComputer**
    - _Uso_: Servidor Edge Industrial.
    - _Features_: ARM Cortex-A72, NVMe SSD, Ethernet Gigabit Dual.
    - _Ideal para_: Gateway central y Edge Computing.

3.  **ReComputer AI**
    - _Uso_: Procesamiento Edge avanzado con aceleración de hardware.
    - _Features_: Integra módulos NVIDIA Jetson para inferencia local.
    - _Ideal para_: Visión artificial centralizada y procesamiento profundo de datos (Deep Learning) en el sitio.
