# Como crear tu perfil para agentes
- Primero crea un archivo en esta carpeta con tu nombre de usuario (Axel, Javier, etc) y con una extension .md (Axel.md, Javier.md).
- Cambia la linea 3 de Base donde dice "(Perfil.md)" y pon el nombre de tu archivo.
- Llena tu archivo con las reglas y guias que quieras que los agentes sigan al trabajar en el proyecto.
